# 💰 SpendSmart — Personal Finance Tracker

> A fully client-side expense tracker with interactive charts, budget alerts, and multi-currency support. Built with pure HTML, CSS & JavaScript — no libraries, no backend, no build step.

## 🌐 Live Demo
🔗 [View Live →](https://spendsmart.vercel.app) <!-- Vercel deploy ke baad update karna -->

---

## ✨ Features

| Feature | Description |
|---|---|
| 📊 Dashboard | Real-time stat cards for Income, Expenses, Net Balance & Savings Rate |
| 🍩 Donut Chart | Pure SVG spending breakdown by category |
| ↕️ Transactions | Full CRUD — add, edit, delete with search & category filters |
| 🎯 Budget Goals | Set a monthly cap with live progress bar & smart alerts at 75% / 100% |
| 📈 Reports | Income vs. Expenses bar chart + top spending categories |
| ⚙️ Settings | Custom categories with emoji, multi-currency, display name |
| 🌙 Dark / Light Mode | Toggle with persistent preference |
| 💾 CSV Export | Download your full transaction history |
| 📱 Responsive | Mobile-friendly sidebar navigation |
| ⌨️ Keyboard Shortcuts | `Esc` to close modals · `Ctrl+Enter` to submit forms |

## 🚀 Getting Started

No installation required. Just open the file:
```bash
git clone https://github.com/saarika-codes/spendsmart.git
cd spendsmart
open index.html
```

---

## 🗂️ Project Structure

spendsmart/
├── index.html     # App shell, all pages & modals
├── style.css      # Full design system (CSS variables, dark mode, components)
└── script.js      # All logic: state, rendering, charts, storage

---

## 🛠️ Tech Stack

- **HTML5** — Semantic structure, ARIA accessibility
- **CSS3** — CSS variables, responsive grid, animations
- **Vanilla JavaScript (ES6+)** — Zero dependencies
- **Web Storage API** — localStorage for data persistence
- **SVG** — Custom donut chart, no chart libraries
